# H5-NES 模擬器

一個使用 HTML5 Canvas + TypeScript 開發的 NES 模擬器。

## 專案目標

在瀏覽器中完整模擬 Nintendo Entertainment System (NES/FC)，支援：
- 完整的 6502 CPU 指令集
- PPU 圖形渲染 (256x240 解析度)
- APU 音頻輸出 (5 聲道)
- 標準控制器輸入
- 常見 Mapper 支援

## 快速開始

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev

# 建置生產版本
npm run build

# 執行測試
npm test
```

## 開發階段

本專案採用分階段開發策略，每個階段都有明確的測試目標：

| 階段 | 目標 | 驗證方式 |
|------|------|----------|
| Phase 1 | CPU 實作 | nestest.nes 測試 ROM |
| Phase 2 | PPU 基礎 | 圖案表顯示測試 |
| Phase 3 | 輸入系統 | 控制器響應測試 |
| Phase 4 | APU 音頻 | 音頻波形測試 |
| Phase 5 | Mapper | 遊戲相容性測試 |
